Se realizo la prueba de desarrollador FullStack con enfoque en Backend de MASTER

Para ver el procedimiento de la prueba, lo primero es clonar el repositorio de github:
Instrucciones para Clonar el Repositorio:

1. Abre tu terminal o línea de comandos.

2. Utiliza el siguiente comando para clonar el repositorio a tu máquina local:
-git clone https://github.com/ingdanielj0816/MasterTest

luego de clonar el repositorio:

Para evaluar el primer test en la terminal dirigirse a la carpeta Test1 y ejecutar la aplicacion de Node:

1. cd MasterTest/Test1

2. node Test1

Para evaluar el segundo test dirigirse a la carpeta Test2 y abrir Test2.js

Para evaluar el tercer test en la terminal dirigirse a la carpeta Test3 y ejecutar la aplicacion de Node:
1. cd MasterTest/Test3

2. node Test3

Para evaluar el cuarto test dirigirse a la carpeta Test4 y abrir Test4.png donde se vera de manera visual el modelado,
tambien dirigirse a Test4.sql, donde estan generadas las tablas mencionadas en el modelado.

Para evaluar el quinto test dirigirse a la carpeta Test5 y abrir Test5.png donde se evidenciara la arquitectura de los 
microservicios de manera visual, tambien dirigirse a Test5.txt donde esta la explicacion de todo el planteamiento de la arquitectura,
asi como el arbol de programas 

Para evaluar el sexto test dirigirse a la carpeta Test6 y abrir Test6.txt donde se encuentra la Política de Nomenclatura para Desarrollo de Software
